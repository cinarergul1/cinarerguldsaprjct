# Creatine Intake and Muscle Growth — Data Analysis Project

##  Project Description
This project looks at how creatine intake may affect muscle growth and strength over a 15-day period. I tracked daily data such as creatine intake, protein intake, sleep, training volume, bench press, and arm size.

The goal is to see if there is any relationship between creatine use and improvements in strength or muscle growth.

---

##  Dataset
The dataset includes 15 days of tracking with the following variables:

- Day number  
- Creatine intake (grams)  
- Protein intake (grams)  
- Sleep duration (hours)  
- Number of sets performed  
- Bench press weight (kg)  
- Arm circumference (cm)  

---

##  What I Did (Methods)
In this project, I used Python to:

- Load and organize the dataset (Pandas)
- Explore the data (EDA)
- Create graphs to see trends (Matplotlib, Seaborn)
- Check relationships between variables (correlation)
- Test hypotheses using statistics (Pearson correlation)
- Run a simple linear regression analysis

---

##  Main Things I Looked At
- How bench press changed over time  
- Changes in arm size over time  
- Relationship between creatine and strength  
- Relationship between protein and performance  
- Overall correlations between all variables  

---

##  Hypothesis Testing
I tested whether creatine intake has a relationship with performance.

- H0 (null hypothesis): Creatine has no effect on performance  
- H1 (alternative hypothesis): Creatine is related to performance  

I used a significance level of 0.05.

---

##  Results (Summary)
From the analysis:

- Bench press increased over time  
- Arm size also showed a small increase  
- Creatine intake showed a positive relationship with strength  
- Protein intake also had some positive relationship with performance  
- Training volume also played an important role  

---

##  Important Note
This is a small dataset (15 days), so results are only suggestive and not final scientific proof.
